# Tugas Visualisasi Data Menggunakan Python

Nama : Nazwa Yulianti Munjana  
Mata Kuliah : Pembelajaran Mesin
Dataset : tips.csv  
Library : Pandas dan Matplotlib  

Pada tugas ini dilakukan visualisasi data dari dataset **tips.csv** dengan menggunakan Python.  
Terdapat dua percobaan visualisasi yang dilakukan untuk memahami pola pada data pelanggan restoran yang memberikan tips.

---

## Percobaan 1: Persentase Laki-laki dan Perempuan yang Memberikan Tips

Pada percobaan pertama dilakukan visualisasi untuk melihat **persentase pelanggan laki-laki dan perempuan** yang memberikan tips di restoran.

Visualisasi dilakukan menggunakan **Pie Chart** sehingga perbandingan antara pelanggan laki-laki dan perempuan dapat terlihat dengan jelas.

## Percobaan 2: Jumlah Pelanggan Berdasarkan Hari

Pada percobaan kedua dilakukan visualisasi untuk mengetahui **jumlah pelanggan yang memberikan tips berdasarkan hari**.

Visualisasi ini menggunakan **Bar Chart** untuk menunjukkan hari dengan jumlah pelanggan paling banyak.

## Kesimpulan

Dari dua percobaan visualisasi yang dilakukan dapat disimpulkan bahwa visualisasi data membantu dalam memahami pola dalam dataset.  
Dengan menggunakan **Pie Chart** dan **Bar Chart**, distribusi pelanggan berdasarkan **gender** dan **hari** dapat terlihat dengan lebih jelas.
